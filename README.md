MyWhatapps

Introduction:
This project is built using claude code with DeepSeek v4 pro. It performs as a web version of whatapps with ai functionality which can talk to ai. Moreover, in Deepseek, i have applied an ai skill , when user send image , video or document to deepseek, it can ask Qwen ai for help and send its result back to the user 

How to use: 
To run this project locally,
1.you have to start it in your terminal first 
2.open your port 8082 in your browser 
3.choose administrator
4.back to MyWhatapps,scan the qr code to link your device 
5.it will show your profile, and list of your contacts
6.you can serach the person you want to chat in the search bar 
7.you can also talk with deepseek , it will always show up in the first row of the list
